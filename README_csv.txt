How to interpret the CSV files
==============================

Each file contains the number of lesions found at each pixel co-ordinate.
Note that this uses the screen standard of (0, 0) located at the top left corner.

For the right eye and composite images:
  Optic nerve position = ( 700 , 550 )
  Macular position = ( 450 , 575 )

For the left eye:
  Optic nerve position = ( 400 , 550 )
  Macular position = ( 650 , 575 )
